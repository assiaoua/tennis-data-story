Tennis Players Graph

Graph

  • This graph groups the ATP tennis match disputed since 2008 with the loser's rank below 40 and the winner's rank below 40.
  • The nodes of the graph correspond to tennis players. The size of the nodes is proportional to the number of matches the player has played.
  • The link between two players groups all the matches that the two players have played together in the given year. The thickness of the links between the players is proportional to the number of matches.

Selection

  • The "Node size" and "Edge size" sliders allow you to set the minimum and maximum size of nodes and links respectively.
  • The "Year" slider allows you to select the matches of a particular year. The size of the nodes and links is adapted for each year.
  • The "Winner Rank range" slider allows you to select only those links between players for which the winner's rank for each match is within the selected range. This deletes only the unselected links and does not vary the size of the nodes and links.

Information

  • To study a particular node, it can be moved by clicking and holding the mouse on it while moving it. When you release the mouse, the node remains where you placed it. You can then vary the year and/or the Winner Rank range to see how the node evolves. To return the node to its initial position, simply click on it.
  • If you move the mouse over a node, information about the player is displayed: name, nationality, starting year as a pro, weight, height and playing hand.
  • If you click on the link between two players, a table with all the matches between the two players for the given year is displayed. This table contains the following information for each match: year, winner, ATP, location, tournament, series, court, surface, round, ATP entry ranking of the match winner or loser as of the start of the tournament (WRank and LRank), ATP entry points of the match winner or loser as of the start of the tournament (WPts and LPts), number of games won in n-th set by match winner or loser (W1, L1, W2, L2, W3, L3, W4, L4, W5 and L5), number of sets won by match winner or loser (Wsets and Lsets), Comment, bet365 odds of match winner or loser (B365W and B365L).
